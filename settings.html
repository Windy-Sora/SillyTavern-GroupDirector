导演剧本 (Director Script)

让导演不仅决定谁发言，还生成一段场景剧本，注入到角色生成 prompt 中指导内容创作。 告诉导演你希望什么样的剧本风格。例如："剧情要温馨治愈，突出姐妹情深"、"保持紧张悬疑的氛围"、"加入搞笑吐槽元素"等。留空则只要求基本场景描述。

连贯剧本模式

选择只注入上一轮计划，还是注入完整历史记录。 指定注入最近 N 轮导演计划的 JSON 数量。设为 0 则注入全部记录。 {{previousPlan}} 占位符会被替换为上一轮导演的完整 JSON 回复。 {{previousPlans}} 占位符会被替换为过往导演计划的 JSON 数组。

世界书注入 (World Info)

将当前激活的世界书/ lorebook 条目注入 Director prompt，让导演了解世界背景设定。 {{worldInfo}} 占位符会被替换为当前激活的世界书条目文本。 控制剧本如何包裹后注入角色 prompt。{{script}} 占位符会被替换为实际剧本内容。
可用占位符：{{recentMessages}}、{{characters}}、{{maxSpeakers}}
模型必须返回 JSON：{"speakers": ["Name1", "Name2"], "reason": "..."}。 启用剧本后还需包含 "script": "..."。 speakers 数组顺序就是发言顺序。
